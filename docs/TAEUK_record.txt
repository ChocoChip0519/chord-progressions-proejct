[FIX 2] songs.json flat/sharp 표기 불일치 → 패턴 매칭 전혀 안 됨
----------------------------------------------------------------
문제
  - music.js의 NOTES 배열은 sharp 표기만 사용 (A#, D#, G# ...)
  - songs.json absolute 필드에는 flat 표기 혼재 (Bb, Eb, Ab, Db ...)
  - getAbsolutePatternRecs에서 문자열 직접 비교
      "A#" !== "Bb" → Fix You, A Thousand Years, Hello 등
      flat key 곡 30여 개가 패턴 매칭에서 전부 탈락

수정 내용 (music.js — getAbsolutePatternRecs)
  - progression.rootNote를 pc(pitch class 0~11)로 변환
  - songs.json absolute 항목도 pc()로 변환 후 숫자 비교
      pc("A#") === pc("Bb") === 10  → 정상 매칭
  - 반환 시 flat 표기를 sharp으로 정규화
      noteFromPc(pc("Bb")) = "A#"

관련 파일
  - src/music.js (getAbsolutePatternRecs 함수 수정)

[FIX 15] Jazz 장르 화성 개선 — 3화음 → 7화음 + 재즈 고유 추천
----------------------------------------------------------------
문제
  - Jazz 장르의 추천 코드와 피아노 코드 모드가 Pop과 동일한
    3화음(triad)을 사용 (C, Dm, G → 재즈답지 않음)
  - transitions.jazz 노드도 "I","ii","V" 등 3화음 표기 그대로
  - parseRoman / chordIntervals가 "maj7" 접미사를 지원하지 않아
    Imaj7(장7도)와 I7(단7도/도미넌트)를 구분할 수 없었음
  - 삼전음 대리코드(bII7), 부속 도미넌트(VI7, II7) 등
    재즈 고유 어법이 전혀 추천되지 않았음

수정 내용

  a) src/music.js — parseRoman
     - "maj7" 접미사 인식 추가 (isMaj7 플래그)
     - "I7"(dominant7) 과 "Imaj7"(major7th) 구분 가능

  b) src/music.js — chordIntervals
     - isMaj7=true 일 때 [0,4,7,11] (장7도) 반환
     - 기존 isSeventh=true 는 [0,4,7,10] (단7도/dominant) 유지

  c) src/music.js — romanToChord
     - isMaj7 전달, 코드 이름 레이블 수정
       "I7" → "C7", "Imaj7" → "Cmaj7"
     - qFinal: "maj7" 타입 추가

  d) src/music.js — getDiatonicRomans
     - Jazz 전용 7화음 다이어토닉 반환 (피아노 스냅용)
       major: ["Imaj7","ii7","iii7","IVmaj7","V7","vi7","viiº"]
       minor: ["im7","iiº","IIImaj7","iv7","V7","VImaj7","VII7"]

  e) src/structures.js — getRecommendations
     - Jazz 전용 예외 코드 추가 (Rock의 bVII와 동일 방식)
       bII7(삼전음 대리), VI7(부속 도미넌트→ii), II7(부속 도미넌트→V)
       다이어토닉 필터를 통과하도록 허용

  f) src/data.js — transitions.jazz
     - 노드 키를 7화음 표기로 전면 교체
     - bII7, VI7, II7 노드 추가 (재즈 고유 진행)
       ex. "ii7" 다음: V7(55%), bII7(20%), IVmaj7(12%), ...
           "VI7" 다음: ii7(70%), V7(20%), IVmaj7(10%)

  g) src/data.js — startingChords / progressions / genres / chordMoods
     - jazz 시작 코드: "I","ii" → "Imaj7","ii7"
     - jazz 대표 진행: ["ii7","V7","Imaj7"], ["Imaj7","vi7","ii7","V7"] 등
     - genres example: "ii–V–I" → "ii7–V7–Imaj7"
     - chordMoods: 재즈 7화음 10개 + 고유 어법 3개 분위기 설명 추가

결과
  - 추천 카드: C major → Cmaj7, Dm → Dm7, G → G7
  - ii7 다음 추천에 Db7(bII7, 삼전음 대리코드) 20%로 등장
  - Imaj7 다음 추천에 A7(VI7, 부속 도미넌트) 10%로 등장
  - Pop/Rock/Blues 동작에는 영향 없음

관련 파일
  - src/music.js (parseRoman, chordIntervals, romanToChord,
                  getDiatonicRomans)
  - src/structures.js (getRecommendations)
  - src/data.js (transitions.jazz, startingChords, progressions,
                 genres, chordMoods)
